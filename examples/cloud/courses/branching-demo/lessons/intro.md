# Branching demo

This example shows how LXPack can **branch** based on variables and results.

## What you’ll try

- Pick a path in an HTML interaction (**sets a variable**)
- See flow rules route you to different activities (**`flow` in `course.yaml`**)
- Finish with a quick quiz and wrap‑up (**completion threshold + assessment gate**)

## Tip

If you want to see branching in action, open `course.yaml` in this example and compare:

- the `variables` section (defaults + types)
- the `flow` rules (`when` → `goto`)
